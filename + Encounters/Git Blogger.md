# Git Blogger

A web application to make it easier to manage contents on GitHub.

Using Expo framework.

设想
- 只需要填写 token
- 用户名可由 token 获取
- 默认仓库名为 [owner].github.io
- 增删改查
- 侧边栏
- 字体设置
- 编辑器
	- 可以像 workflowy一样对整行进行删除、上下拖拽
